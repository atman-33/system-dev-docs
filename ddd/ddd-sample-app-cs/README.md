# DDDサンプルアプリ(C#)

## プロジェクト構成

```txt
+- DDDSampleApp.Domain          : ドメイン層
|
+- DDDSampleApp.Infrastructure  : インフラ層 
|
+- DDDSampleApp.UseCase         : ユースケース層（アプリケーション層）
|
+- DDDSampleApp.Wpf             : プレゼンテーション層（Wpf + Blazor Hybrid）
|
+- DDDSampleAppTest.Tests       : テストプロジェクト
```

## 利用技術スタック

- WPF
- Blazor Hybrid
- TailwindCSS
- EntityFramework Core
